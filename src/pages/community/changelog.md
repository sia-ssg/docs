# Changelog

All notable changes to Sia will be documented here.

## Version 2.2.0

### Added

- **Plugin System**: Powerful new plugin system that allows extending Sia at key points in the build lifecycle
  - Support for local plugins in `_plugins/` directory
  - Support for npm package plugins with `sia-plugin-*` naming convention
  - Comprehensive hook system for build lifecycle, content processing, and template customization
  - Plugin API with utilities for file operations, logging, and configuration access
  - Plugin configuration system with per-plugin settings
  - Plugin dependency management
  - See [Plugins](/plugins/) for complete documentation
