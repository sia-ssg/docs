# Contributing

Thank you for your interest in contributing to Sia!

## How to Contribute

1. **Report Issues** - Found a bug? [Open an issue](https://github.com/terrymooreii/sia/issues)
2. **Suggest Features** - Have an idea? [Start a discussion](https://github.com/terrymooreii/sia/discussions)
3. **Submit Pull Requests** - Fixed a bug or added a feature? Submit a PR!

## Development Setup

1. Fork the repository
2. Clone your fork
3. Install dependencies: `npm install`
4. Make your changes
5. Test your changes
6. Submit a pull request

## Code Style

- Follow existing code style
- Add tests for new features
- Update documentation as needed

## Questions?

Feel free to [open an issue](https://github.com/terrymooreii/sia/issues) or [start a discussion](https://github.com/terrymooreii/sia/discussions).
