Welcome to Sia! This guide will help you get started with Sia, a simple and powerful static site generator.

## What is Sia?

Sia is a static site generator built with JavaScript, similar to Eleventy/11ty. It supports:

- **Markdown** with YAML front matter
- **Nunjucks templates** for flexible layouts
- **Multiple content types** (posts, pages, notes)
- **Tags and categories** for organization
- **Built-in themes** with dark mode support
- **Live reload** development server
- **RSS feed** generation

## Quick Start

1. **Install Sia globally:**
   ```bash
   npm install -g @terrymooreii/sia
   ```

2. **Create a new site:**
   ```bash
   sia init my-blog
   ```

3. **Start the development server:**
   ```bash
   cd my-blog
   npm run dev
   ```

4. **Visit your site:**
   Open `http://localhost:3000` in your browser.
